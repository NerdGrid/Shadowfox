# 🏏 Advanced Task – Cricket Fielding Analysis

## ShadowFox Data Science Internship

### 📌 Project Overview
This project analyzes the fielding performance of three cricket players from a T20 match. The analysis evaluates fielding actions such as clean picks, catches, throws, direct hits, run-outs, and runs saved to measure each player's overall defensive contribution.

### 🎯 Objectives
- Collect fielding data for selected players.
- Calculate individual fielding performance.
- Analyze different fielding actions.
- Compare player performances using statistical analysis.
- Present insights through visualizations.
